# Description

Linux kernel driver for controlling [Clevo](https://www.clevo.com.tw/clevo_pro.asp?lang=en "Clevo Product Page") keyboard backlight

# Installation

1. make
2. sudo make install

## Dependencies
- make
- gcc
- linux-headers
